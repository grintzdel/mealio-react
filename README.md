# Single page React + Vite

Single page simpliste d'un site de recettes de cuisine.

## Installation
### AVOIR NODE D'INSTALLÉ !!

```
git clone https://github.com/grintzdel/mealio-react.git
```

```
cd mealio-react
```

```
npm i 
```

## Développement et serveur local 

```
npm run dev
```

## Build

```
npm run build
```
